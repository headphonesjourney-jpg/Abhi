# NXTGEN Trading System Landing Page

## Description
High-conversion fintech landing page built using HTML5, Tailwind CSS CDN, and vanilla JavaScript.

## Features
- Mobile-first responsive design
- Sticky header with CTA
- WhatsApp floating button
- Smooth scrolling
- SEO & Open Graph optimized

## Deployment
1. Upload all files to any static hosting (Netlify, Vercel, GitHub Pages)
2. Open index.html in browser

## GitHub Upload
1. Create a new repository
2. Upload all files
3. Enable GitHub Pages (Settings → Pages)
